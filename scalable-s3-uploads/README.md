# Scalable S3 uploads

## Requirements

- [Bun](https://bun.sh)

## Installation

```shell
bun install
```

## Running tests

```shell
bun run test
```

## (Optional) Generate a sample csv file

In the root of the project run:

```
bun scripts/create-huge-file.ts
```

Quickly you'll see a file named `largefile.csv` in the same `scripts` folder.
